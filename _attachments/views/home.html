What's going on here? This is a simple study in non-repeated form, shape, color and animation. Art from code, code from art. The Processing programming language and Processing.js Javascript extension are used to generate unique renderings based on template sketches. Lots of other neat-o technologies and libraries are being used here too, like CouchDB and Sammy.
